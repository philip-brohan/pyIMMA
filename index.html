A Python API for IMMA dataÂ¶

The observations in the International Comprehensive Ocean-Atmosphere DataSet (ICOADS), are in the International Meteorological Maritime Archive (IMMA) format. This package provides a Python API for using data in the IMMA format.

This is not an official ICOADS product. There is official software for IMMA, but only in Fortran.

This software supports all of IMMA version 0, and all main version 1 records (those starting with a core extension). Subsiduary records are not yet supported.
