How to use this dataset¶

This dataset is kept under version control in a git repository. The repository is hosted on GitHub (and the documentation made with GitHub Pages). The repository is https://github.com/philip-brohan/pyIMMA.

If you are familiar with GitHub, you already know what to do: If you’d prefer not to bother with that, you can download the whole dataset as a zip file.

To use this module, clone or download the repository and then install it with:

python setup.py install --user
